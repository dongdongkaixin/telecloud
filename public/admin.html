🔌 WebDAV 挂载说明

TeleCloud 支持标准 WebDAV 协议，可通过以下方式挂载到本地：

📋 WebDAV 连接信息

服务器地址：https://your-pages-domain.pages.dev/api/webdav/ 用户名：您的管理员用户名 密码：您的管理员密码

🖥️ Windows 挂载（映射网络驱动器）

1. 打开"此电脑"→"映射网络驱动器" 2. 文件夹填写：https://your-domain.pages.dev/api/webdav/ 3. 勾选"使用其他凭据连接" 4. 输入用户名和密码

🍎 macOS 挂载

1. Finder → 前往 → 连接服务器 2. 输入：https://your-domain.pages.dev/api/webdav/ 3. 输入用户名和密码

📱 Alist 挂载配置

存储类型：WebDAV 服务器地址：https://your-domain.pages.dev/api/webdav/ 用户名：管理员用户名 密码：管理员密码 根文件夹路径：/

☁️ Rclone 配置

[telecloud] type = webdav url = https://your-domain.pages.dev/api/webdav/ vendor = other user = admin pass = your_password
